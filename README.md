# Modular-Calculator-Task-4
Modular Calculator in Java


## Description
This project is a Java modular calculator program designed to demonstrate
method design and clean coding practices.

## Features
- Separate methods for each calculator operation
- Method overloading using different parameter types
- Modular and reusable code structure
- Exception handling for divide-by-zero
- Demonstration of pass-by-value in Java

## Operations Implemented
- Addition
- Subtraction
- Multiplication
- Division

## Tools Used
- Java
- IntelliJ IDEA / Eclipse
